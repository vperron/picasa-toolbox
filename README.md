picasa-toolbox
==============

A set of apis and tools to manage your pictures, both locally
and online on Google+ Photos.


Customization quick start
=========================


```bash
pip install -e .[test]
py.test
```

Finally, give the command line program a try.

```bash
ptoolbox --help
```
